# Choppa's Meme Library

Original content memes, classified by state.

A library, not a dumping ground. Every meme here is OC (made by Chop) — repurposed templates, community reactions, Pyth-specific gags, broader crypto satire. If it isn't original, it doesn't belong.

## By State

- [[wip/index|WIP]] — memes with potential, not yet finished or shipped
- [[done/index|Done]] — finished, posted memes (with deployment context where worth remembering)

## Conventions

| Rule | Why |
|------|-----|
| OC only | The library is a portfolio, not a reaction folder |
| Lowercase-kebab-case filenames | So we can reference them in markdown without escape hell |
| `.png` preferred for memes with text | Crisp edges on typography |
| `.jpg` preferred for photo-base memes | Smaller file size, irrelevant artifacts |
| Move → don't copy | When a WIP ships, move the file to `done/` rather than duplicating |

## Transitions

A meme's journey:

1. **Seed** — drop the raw image into `wip/`
2. **Refine** — tweak copy, test with the group, iterate
3. **Ship** — post it publicly (Twitter, Discord, Telegram, wherever)
4. **Archive** — move the file to `done/`, note the deployment (platform + date + vibe-read of how it landed)

Dead memes (tried, didn't land, not worth reviving) — delete them. The library is curated; there's no "failed/" folder.
